PROJECTS

Raspberry Pi Rover

In my first project, I explore the fusion of robotics and programming with the Raspberry Pi Rover, a sophisticated and versatile mobile robot. This project showcases the power of Raspberry Pi in controlling and navigating a mars-style rover, capable of traversing diverse terrains. Equipped with a camera for real-time video feedback, this rover is a testament to innovative engineering and hands-on IoT development.

Lunchbox Computer

Redefining portability, the Raspberry Pi Lunchbox Computer is a compact, powerful, and fully functional computer housed in a creatively repurposed lunchbox. This project is a perfect blend of technology and practicality, featuring a Raspberry Pi at its core, complete with a bluetooth keyboard, integrated power supply, and a user-friendly interface. Ideal for on-the-go computing, it demonstrates ingenuity in compact hardware design.

Raspberry Pi Cluster

This groundbreaking project combines the versatility of Raspberry Pi with the power of Jetson Nano in a cluster computing setup. The OctoPi with Jetson Nano cluster is designed for high-performance tasks, showcasing distributed computing and efficient processing capabilities. This setup is ideal for complex computations, machine learning tasks, and large-scale data processing, demonstrating an innovative approach to building powerful, yet cost-effective, computing clusters.

Hangman Game

I dive into the world of classic word games with this Python Hangman Game. Designed with an intuitive command line interface, this game challenges players to guess words, one letter at a time, with a limited number of guesses. Featuring a rich vocabulary database, this project not only entertains but also showcases a practical application of Python programming, project planning and algorithm design.

Tic-Tac-Toe Game

This is my own version of the classic Tic-Tac-Toe game, reimagined with an intelligent twist. This Python-based project introduces an AI opponent, leveraging algorithmic strategies to provide a challenging and unpredictable gaming experience. This project exemplifies the integration of artificial intelligence in game development using Python.

Chicken Image Recognition Model

This project stands at the forefront of AI and machine learning, featuring a sophisticated image recognition model tailored for chickens. Utilizing Pytorch, a leading deep learning framework, the model is trained on a diverse dataset to accurately identify chickens from images. This project demonstrates the power of neural networks in practical applications and offers insights into the capabilities of AI in the image recognition field.
